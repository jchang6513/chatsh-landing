[ BETA v0.1.3 ]

chat.sh

Terminal-native desktop app for managing AI coding assistants.
Claude Code, Codex, Gemini CLI — all in one window. ~4MB, no Electron.

MULTI-REPL
Claude Code, Codex, Gemini, Zsh, Python, Node — side by side
TEMPLATES
Auto-detects installed CLIs. Save custom templates with system prompts
SYSTEM PROMPTS
Per-REPL CLAUDE.md / GEMINI.md / AGENTS.md, fully isolated
SHELL TABS
Add, rename, close shell tabs per panel. Stable numbering
KEYBOARD FIRST
⌘K palette, ⌘1-9, ⌘T/W shell tabs, ⌘R restart, ⌘, prefs
THEMES
8 color schemes synced to terminal ANSI colors. Amber default

KEYBOARD SHORTCUTS

⌘1–9Switch to REPL
⌘[ / ⌘]Previous / Next REPL
⌘KCommand Palette
⌘NNew REPL
⌘RRestart REPL
⌘T / ⌘WNew / Close shell tab
⌘Shift+[ / ]Switch shell tab
⌘,Preferences
EscClose overlay

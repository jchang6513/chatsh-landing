[ BETA v0.1.7 ]

chat.sh

Terminal-native desktop app for managing AI coding assistants.
Claude Code, Codex, Gemini CLI — all in one window. ~4MB, no Electron.

INSTALL

[↓ macOS Apple Silicon .dmg]
# Homebrew
brew tap jchang6513/chatsh
brew install --cask chatsh
# If macOS says "app is damaged"
xattr -cr /Applications/chat.sh.app
SESSION PERSISTENCE
Panes survive app restart via background daemon. Claude, Gemini, Codex sessions keep running even when app is closed
MULTI-PANE
Claude Code, Codex, Gemini, Zsh — side by side. Right-click to Edit / Duplicate / Restart
TEMPLATES
Built-in templates for Claude, Gemini, Codex, Zsh. Add custom templates with system prompts
SYSTEM PROMPTS
Per-Pane CLAUDE.md / GEMINI.md / AGENTS.md, fully isolated
SHELL TABS
Shell tabs per Pane. Auto-scroll to active. Clock + battery in status bar
KEYBOARD FIRST
⌘K palette, ⌘1-9, ⌘T/W shell tabs, ⌘= zoom, ⌘, prefs. 11 color schemes

KEYBOARD SHORTCUTS

⌘1–9Switch to Pane
⌘[ / ⌘]Previous / Next Pane
⌘KCommand Palette
⌘NNew Pane
⌘RRestart Pane
⌘T / ⌘WNew / Close shell tab
⌘Shift+[ / ]Switch shell tab
⌘,Preferences
EscClose overlay
